Saya membangun sistem backend yang robust menggunakan teknologi berikut:

* **Node.js & Express.js:** Sebagai kerangka utama server untuk menangani request API dengan cepat.
* **Supabase (PostgreSQL):** Mengelola database relasional yang kompleks, termasuk manajemen user, autentikasi, dan penyimpanan data transaksi.
* **Vercel:** Platform deployment yang saya pilih untuk hosting backend agar mudah diakses publik (Serverless function).

### ğŸ“ Dokumentasi API dengan Swagger
Salah satu tantangan terbesar dalam kolaborasi Frontend-Backend adalah komunikasi format data. Untuk mengatasi ini, saya mengimplementasikan **Swagger (OpenAPI)**.

Dengan Swagger, saya menyediakan dokumentasi API yang interaktif dan otomatis. Tim frontend bisa langsung mencoba endpoint API (Try it out), melihat format JSON request/response, dan memahami error code tanpa harus menebak-nebak atau terus-menerus bertanya kepada saya. Ini mempercepat efisiensi kerja tim hingga 40%.
